## 一、先立两个预期

**第一，注册是免费的。** 注册和订阅是两道独立的墙：这一篇只办账号，一分钱不花；付费是下一篇（订阅与支付）的事。别被某些二手教程"注册就要买号"的说法带偏——**自己注册，不要买号**，来路不明的共享号、成品号最容易连锁封号。

**第二，注册 ChatGPT 当前不需要手机号。** 新设备或异常位置登录时，系统可能要求邮箱验证码，或者让你在已经登录的 ChatGPT 手机 App 中确认。先照主路径注册，遇到额外验证再按第四节处理。

## 二、开始前的两样准备

**1. 梯子开着，且上一篇验收全绿。** 注册是 OpenAI 风控最敏感的时刻：IP 干不干净，直接决定你是一路顺畅还是反复被验证。开工前用 ping0.cc 再确认一遍：归属地在支持列表内、风险值低、TUN 模式开着。

**2. 一个自己长期控制的邮箱。** 它以后还要用来收登录验证码和找回账号，不要使用临时邮箱。Gmail、Outlook 等邮箱可以直接使用；如果页面提供 Google、Microsoft 或 Apple 登录，也可以选择自己最方便的一种。

- 没有合适邮箱时，可以先在 `outlook.com` 创建一个长期使用的 Outlook 邮箱。
- 已有 Gmail、Microsoft 或 Apple 账号时，可以使用页面提供的快捷登录方式。

## 三、实操：注册 OpenAI 账号

1. 打开 `chatgpt.com`，点 **Sign up**（注册）。如果页面显示异常，再换无痕窗口排除旧缓存干扰
2. 选一条路：
   - **快捷登录**：选择页面显示的 Google、Microsoft 或 Apple 登录，并完成授权。
   - **用邮箱注册**：输入邮箱，按页面提示设置账号并完成邮箱验证。
3. 按提示填姓名、生日。**如实填**，生日用于年龄核验，成年人不会有任何问题
4. 看到聊天界面 = 注册成功。随便发一句话试试，有回复就说明账号是活的

## 四、遇到额外登录验证怎么办

OpenAI 官方当前说明：新建账号和使用 ChatGPT 不要求手机验证。本教程也不使用 API key，因此手机号不在主流程里。

登录新设备或系统认为位置异常时，可能出现额外验证：

1. **邮箱验证码**：去注册邮箱查看一次性验证码，确认发件域名属于 OpenAI 官方域名。
2. **手机 App 确认**：如果你已经在 ChatGPT 手机 App 登录，可以在手机上批准这次登录。
3. **一直收不到验证**：先检查垃圾邮件，再重新发送；仍失败时换一个稳定节点和无痕窗口重试。
4. **页面突然要求手机号码**：先确认网址确实属于 OpenAI。不要使用来路不明的一次性号码；本教程不需要为了 API key 完成手机验证。

## 五、注册时的常见报错

| 现象                               | 原因                  | 解法                              |
| -------------------------------- | ------------------- | ------------------------------- |
| 提示不支持你所在的地区 / Signup unavailable | 出口 IP 位于未支持地区，或当前连接异常 | 换一个支持地区的稳定节点，再用无痕窗口重试                    |
| 人机验证没完没了地转圈                      | 网络、缓存或风控没有完成         | 换节点，清理缓存或使用无痕窗口                          |
| 验证码邮件收不到                         | 进了垃圾箱 / 延迟          | 翻垃圾邮件文件夹，等 1–2 分钟再点重发           |
| 提示邮箱已被注册                         | 以前注册过忘了             | 直接走 Log in → Forgot password 找回 |

## 六、这一篇做完的标志

- [ ] chatgpt.com 能登录，看到聊天界面
- [ ] 发一句话，AI 有回复
- [ ] 记好你的登录方式（Google 授权 or 邮箱+密码），下一篇订阅、下下篇 App 登录都靠它

注意：注册完成后通常是一个 **Free 账号**。Free 当前可以短暂试用 Codex，但较低的使用额度不是这套教程的完成基线；下一步需要把账号升级为 Plus 或更高付费套餐。如果账号已经是更高付费套餐，只需确认套餐状态。
