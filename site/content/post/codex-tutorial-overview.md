## 一、先看它能帮你做什么

**Codex 不只是回答问题，它还能在你允许的文件夹里真正完成任务。**

这里的 **BL75** 是我们参加“生活攻略”第 75 期课程时组成的学习团体。文中所说的 `BL75` 文件夹，是这个团体在课程期间整理的课程笔记、功课题目、自我指引工具和组会规则。

BL75 的公开资料库地址是 [https://github.com/ken-zy/BL75](https://github.com/ken-zy/BL75)。第 5 篇会提供可以直接交给 Codex 的提示词，让它检查并安装 Git，再把这份公开资料连同版本记录一起复制到你的电脑。

例如：

- 你说“整理 BL75 资料，先告诉我应该怎样分类”，它可以阅读现有文件，提出整理方案，等你确认后再修改。
- 你说“帮我做一个简单的个人网页”，它可以创建文件、生成页面，并把结果交给你检查。
- 你说“检查这个项目哪里有问题”，它可以阅读项目、解释原因、修改文件，再汇报具体改了什么。

这套教程不会一次介绍所有高级功能。我们只完成一个真实闭环：**从 GitHub 把 BL75 clone 到电脑，用 Codex 阅读它；再用自己的测试文件夹练习 Git 和 GitHub，最后用 Obsidian 把 BL75 变成个人知识库。**

## 二、Codex 是什么

**Codex 是 OpenAI 的 AI 软件开发智能体。**“智能体”可以理解为：它不只给建议，还能在获得权限后阅读文件、执行操作并交付结果。

你不需要先学会编程，但需要承担三件事：

1. 说清楚想要什么结果。
2. 在修改、上传和公开之前作出确认。
3. 检查 Codex 交付的结果是否正确。

对零基础读者来说，最直观的入口是 ChatGPT 桌面 App 里的 Codex。当前官方快速入门是：安装桌面 App、登录 ChatGPT 账号、选择项目或本地文件夹，再发送第一条任务。

## 三、大陆用户开始前必须知道的风险

OpenAI 当前的 ChatGPT 支持地区列表中不包含中国大陆、香港和澳门。官方同时提醒：从支持列表之外访问或提供访问，可能导致账号被封禁或暂停。

因此，网络篇介绍的是实际连接和排查方法，不代表 OpenAI 对未支持地区作出了服务承诺。继续之前，请自行确认符合所在地法律以及你愿意接受的账号风险。

## 四、怎么读这份教程

主路线一共六步：

1. 网络准备。
2. 注册并验证自己的 ChatGPT 账号。
3. 订阅 Plus，或确认已有更高付费套餐。
4. 安装桌面 App，打开一个不含隐私的本地测试文件夹。
5. 让 Codex 从 GitHub clone BL75，并用个人测试文件夹练习 Git 和 GitHub。
6. 用 Obsidian 打开已经 clone 的 BL75，再让 Codex 按来源回答。

第 3 篇“订阅与支付”是主路线的必经步骤。Free 当前可以打开并短暂试用 Codex，但这套教程还要连续完成安装、本地文件读取、版本管理和知识库练习；开始第 4 篇前，需要升级 Plus 或确认账号已有更高付费套餐。

## 六、开始前必须知道的三件事

1. **Free 当前可以试用 Codex，但不是本教程的完成基线。** 不同套餐的使用额度不同；本教程按 Plus 或更高付费套餐准备。
2. **注册 ChatGPT 当前不要求手机验证。** 新设备或异常登录可能触发邮箱验证码或移动端确认；本教程不使用 API key。
3. **本地文件是否公开，由你决定。** Codex 可以阅读本地文件，不等于这些文件会自动出现在 GitHub。创建公开仓库和上传都必须再次确认。

## 七、成本总账

- **必需预算**：一条可用的网络线路，以及至少一个月 ChatGPT Plus。
- **Plus 价格**：官方价格为 `$20/月`，实际本币金额和税费以结账页或应用商店显示为准；已有更高付费套餐则不需要重复购买 Plus。
- **Git、GitHub 公开仓库和 Obsidian 基础功能**：完成本教程不需要额外付费。

先完成账号注册，再只通过官方入口订阅，避免同时购买与本教程无关的服务。

## 八、一句话安全提醒

账号必须由自己注册和保管；不要购买共享账号或把密码交给代充。任何准备上传 GitHub 的文件，都要先检查密码、密钥、私人笔记和他人隐私。
